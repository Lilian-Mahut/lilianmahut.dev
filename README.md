## Running Locally


```sh-session
git clone https://github.com/Lilian-Mahut/lilianmahut.dev.git
cd lilianmahut.dev
```


Create a `.env` file similar to [`.env.example`](https://github.com/Lilian-Mahut/lilianmahut.dev/blob/main/.env.example).

Then install dependencies and run the development server:
```sh-session
pnpm install
pnpm dev
```
